# Commits em Português

Ao sugerir ou gerar mensagens de commit para o usuário, **sempre use português brasileiro**.

## Formato padrão:
```
<tipo>: <descrição em português>
```

## Tipos comuns:
- `feat:` nova funcionalidade
- `fix:` correção de bug
- `style:` ajuste visual/CSS
- `refactor:` refatoração de código
- `docs:` documentação
- `chore:` tarefas de manutenção

## Exemplos:
- `feat: adicionar opções de cartão crédito e débito`
- `fix: corrigir erro de TypeScript na página de configuração`
- `style: aumentar tamanho da imagem de produto no mobile`
- `refactor: migrar APIs do Firebase para Supabase`
